# Long Building

A relaxing-then-tense block-stacking arcade game for iOS.

Stack houses on top of houses with a crane that swings against the sky — from a
peaceful start all the way to the edge of space. Time the crane, land each block
square, and chase perfect placements as the music builds.

## Support

Have a question, found a bug, or want to suggest a feature?
Open an [issue](https://github.com/fatih-koca/long-building-app/issues) on this
repository and we'll take a look.

## Privacy

Long Building does not collect or share any personal data of its own. The game
runs fully offline and stores all progress on your device. The app does show
Google AdMob ads. See the full **[Privacy Policy](PRIVACY.md)** for details.
